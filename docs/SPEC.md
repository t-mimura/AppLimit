# AppLimit - アプリ利用制限アプリ 仕様書

## 概要

特定のアプリ（主にSNS）の連続使用時間を制限するAndroidアプリ。
設定した制限時間が経過すると全画面オーバーレイで警告を表示し、対象アプリの操作を遮断する。

## 対象ユーザー

- 開発者本人（1名）
- Google Play Store に公開する

## 用語定義

| 用語 | 定義 |
|------|------|
| 対象アプリ | ユーザーが制限対象として登録したアプリ |
| セッション | 対象アプリが前面に表示されてから制限時間に達するまでの一連の使用期間 |
| 制限時間 | セッション開始からオーバーレイ表示までの時間。対象アプリごとに設定可能 |
| クールダウン | 制限発動後、次のセッションを開始できるようになるまでの待機時間 |
| 延長 | 制限発動時に制限時間を一時的に追加する操作 |
| オーバーレイ | 対象アプリの上に表示される全画面の警告画面 |

## 機能要件

### F1: 対象アプリの管理

- インストール済みアプリの一覧から複数の対象アプリを選択・登録できる
- 対象アプリごとに以下を個別設定できる:
  - 制限時間（デフォルト: 15分、1分単位で設定可能）
  - クールダウン時間（デフォルト: 2時間、15分単位で設定可能）
  - 延長時間（デフォルト: 5分、1分単位で設定可能）
  - 制限の有効/無効トグル
- 対象アプリの登録解除ができる

### F2: 使用時間の監視

- バックグラウンドサービスが常駐し、対象アプリのフォアグラウンド状態を監視する
- 対象アプリがフォアグラウンドに来た時点でセッションタイマーを開始する
- タイマーはセッション中は対象アプリがバックグラウンドに移行しても停止しない（簡易実装方針）
  - 理由: フォアグラウンド/バックグラウンドの切り替えの正確な追跡は複雑であり、初回リリースではシンプルな実装を優先する
- 制限時間に達したらオーバーレイを表示する

### F3: オーバーレイ警告

- 制限時間到達時に対象アプリの上に全画面オーバーレイを表示する
- オーバーレイには以下を表示する:
  - 「制限時間に達しました」等のメッセージ
  - 対象アプリ名
  - 使用した時間
  - 「延長する」ボタン
  - 「閉じる（ホームに戻る）」ボタン
- オーバーレイ表示中は対象アプリの操作ができない

### F4: 延長

- オーバーレイの「延長する」ボタンを押すと、設定された延長時間分だけセッションが延長される
- 延長回数に制限はない（v1）
- 延長後、再度制限時間に達したら再びオーバーレイを表示する

### F5: クールダウン

- ユーザーがオーバーレイの「閉じる」を選択した場合、セッションが終了しクールダウンに入る
- クールダウン中に対象アプリを開いた場合、即座にオーバーレイを表示する
  - オーバーレイには「クールダウン中です（残り○分）」と表示する
  - クールダウン中も「延長する」ボタンは利用可能（延長時間分だけ使用できる）
- クールダウン時間が経過すると、新しいセッションとして通常利用できる

### F6: 通知

- 制限時間の残り5分前（制限時間が5分以下の場合は残り1分前）に通知を出す
- 制限発動時に通知を出す

## 非機能要件

### NF1: パーミッション

以下のパーミッションを使用する:

- `SYSTEM_ALERT_WINDOW` — オーバーレイ表示
- `PACKAGE_USAGE_STATS` — アプリ使用状況の取得
- `FOREGROUND_SERVICE` — バックグラウンド監視サービス
- `RECEIVE_BOOT_COMPLETED` — 端末再起動後のサービス自動起動
- `POST_NOTIFICATIONS` — 通知の表示（Android 13+）

### NF2: バッテリー消費

- 監視ポーリング間隔を適切に設定し、過度なバッテリー消費を避ける
- ポーリング間隔: 3秒（対象アプリのフォアグラウンド検出精度とバッテリーのバランス）

### NF3: 対応バージョン

- minSdk: 26 (Android 8.0)
- targetSdk: 35 (Android 15)
- compileSdk: 35

### NF4: データ保存

- すべての設定・状態はローカル（端末内）に保存する
- サーバー通信は行わない

## 画面構成

### S1: メイン画面

- 登録済み対象アプリの一覧表示
  - アプリアイコン、アプリ名、制限時間、有効/無効トグル
- 「アプリを追加」ボタン → S2へ

### S2: アプリ選択画面

- インストール済みアプリの一覧をリスト表示（検索フィルタ付き）
- タップで選択 → S3へ

### S3: アプリ個別設定画面

- 制限時間の設定
- クールダウン時間の設定
- 延長時間の設定
- 有効/無効トグル
- 削除ボタン

### S4: オーバーレイ画面

- 全画面表示
- 警告メッセージ
- 「延長する」ボタン
- 「閉じる」ボタン
